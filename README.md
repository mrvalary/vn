# vn

`vn` - консольный клиент для заметок, мониторинга устройств и просмотра событий безопасности в ИТ-инфраструктуре VPN-сервиса.

## Назначение

Система позволяет:

- авторизоваться или зарегистрироваться в консольном клиенте;
- создавать, смотреть, редактировать, удалять и искать заметки на устройстве;
- проверять наличие обновлений и запускать установщик обновления;
- получать статистику CPU, RAM и HDD с устройств через агент `VnWatcher`;
- смотреть журнал безопасности;
- разделять доступ по ролям `user`, `admin`, `statistician`.

## Запуск

Основное приложение запускается как консольный клиент:

```powershell
.\CursovoyProjectxDxD.exe
```

При запуске приложение сначала проверяет подключение к базе данных, затем предлагает войти или зарегистрироваться. После успешного входа открывается интерактивная консоль `vn>`.

## YAML-конфигурация

Безопасные настройки вынесены в YAML-файлы рядом с `.exe`. Пароли и строки подключения остаются в `App.config`.
У Watcher-а в `App.config` хранится только строка подключения `WatcherDb`.

`vn.yml` управляет настройками проверки обновлений:

```yaml
updates:
  owner: mrvalary
  repo: vn
  assetExtension: .zip
```

`watcher.yml` управляет настройками агента мониторинга:

```yaml
watcher:
  deviceKey: ""
  deviceName: ""
  intervalSeconds: 60
  databaseTimeoutSeconds: 10
```

Если YAML-файл отсутствует, приложение использует значения по умолчанию.

## Авторизация

На стартовом экране доступны действия:

- `1` - войти;
- `2` - зарегистрироваться;
- `Esc` - выйти.

После входа приложение работает от имени роли пользователя. Доступные команды зависят от роли.

## Карта команд

### Основные

| Команда | Описание |
| --- | --- |
| `help` | Показать карту команд |
| `version` | Показать текущую версию |
| `auth logout` | Выйти из текущей учетной записи |
| `exit` | Закрыть приложение |

### Заметки

| Команда | Описание |
| --- | --- |
| `nt add "текст"` | Добавить заметку |
| `nt list` | Показать свои заметки |
| `nt edit <id> "новый текст"` | Изменить свою заметку |
| `nt del <id>` | Удалить свою заметку |
| `nt search "текст"` | Найти свои заметки |

Пример:

```text
vn> nt add "Проверить конфигурацию VPN-шлюза"
vn> nt list
vn> nt search "VPN"
```

### Мониторинг устройств

Команды доступны ролям `admin` и `statistician`.

| Команда | Описание |
| --- | --- |
| `watch list` | Показать устройства мониторинга |
| `watch show <deviceKey> [count]` | Показать последние метрики CPU/RAM/HDD |
| `watch add <deviceKey> <name> [address] [description]` | Добавить или обновить устройство |
| `watch del <deviceKey>` | Удалить устройство из мониторинга |

Пример:

```text
vn> watch list
vn> watch show MRVALARYPC 10
```

## Watcher-агент

`VnWatcher` собирает нагрузку устройства и отправляет данные в базу данных.

По умолчанию агент:

- собирает CPU, RAM и HDD;
- отправляет данные раз в 60 секунд, если в `watcher.yml` не указан другой интервал;
- сам создает устройство в таблице мониторинга, если такого `deviceKey` еще нет;
- подключается к базе через отдельную учетную запись `vn_watcher`.

Примеры запуска:

```powershell
.\vn-watcher.exe
```

## Логи безопасности

Команды доступны администратору.

| Команда | Описание |
| --- | --- |
| `sec logs [count]` | Показать последние события безопасности |

Пример:

```text
vn> sec logs 20
```

## Администрирование

Команды доступны только роли `admin`.

| Команда | Описание |
| --- | --- |
| `admin user create <login> <password> [user\|admin\|statistician]` | Создать пользователя |
| `admin user list` | Показать пользователей |
| `admin user info <login>` | Показать данные пользователя |
| `admin user block <login>` | Заблокировать пользователя |
| `admin user unblock <login>` | Разблокировать пользователя |
| `admin user delete <login>` | Удалить пользователя |
| `admin user nt <login>` | Показать заметки пользователя |
| `admin nt view <id>` | Посмотреть любую заметку |
| `admin nt edit <id> "новый текст"` | Изменить любую заметку |

## Обновления

При запуске приложение проверяет наличие новой версии. Для ручного управления доступны команды:

| Команда | Описание |
| --- | --- |
| `update check` | Проверить наличие обновления |
| `update apply` | Запустить установщик обновления |

## Роли

| Роль | Возможности |
| --- | --- |
| `user` | Работа со своими заметками |
| `statistician` | Просмотр и управление мониторингом |
| `admin` | Все команды, включая пользователей, заметки, мониторинг и логи безопасности |
